# 基于springboot实现的中/后台脚手架

### 项目介绍

该项目后端使用springboot框架，前端使用vue2和ant-design-vue1.7.8

### 部署说明

(1)将“数据库和maven”目录下数据库导入到本地

(2)将“数据库和maven”目录下“leaf”目录为项目所需的maven依赖，将此目录放在maven仓库的根目录下面即可
